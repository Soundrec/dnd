		папка dnd

Для работы необходимо создать виртуальную среду (язык Python 3.9) и установить в нее библиотеки. 
В терминале (лучше всего Anaconda):
#Создание среды
conda create -n [env_name]
#Переход в среду
conda activate [env_name]
#Переход в директорию проекта
cd [dir]
#Установка библиотек
pip install transformers==4.33.2
pip install torch==2.2.0
pip install sdkit==2.0.15
pip install gitPython==3.1.42
pip install pillow==9.5.0
pip install django==4.2.11
pip install django-simple-captcha==0.6.0
pip install djangorestframework==3.15.1

#Если библиотеки установятся неправильно (какие-то версии могут не совпасть), следуйте инструкциям терминала
#Если какие-то зависимости не установятся или будут удалены при установке, так же следуйте инструкциям терминала

#Запуск локального сервера
python manage.py runserver

#Если необходимо вносить изменения в БД, создайте суперпользователя
python manage.py createsuperuser
#Далее следуйте инструкциям терминала по созданию суперпользователя
